# CodeAlpha Data Analytics Internship

This repository contains projects completed as part of the CodeAlpha Data Analytics Internship.

## Tasks Completed

### Task 1 – Web Scraping
- Extracted book data from a public website using BeautifulSoup.
- Created a custom dataset.
- Performed analysis and visualizations.

### Task 2 – Exploratory Data Analysis (EDA)
- Analyzed global layoff trends from 2020–2025.
- Identified key patterns across industries, companies, and countries.
- Generated business insights using Python.

### Task 3 – Data Visualization
- Designed an interactive Cybersecurity Dashboard in Power BI.
- Monitored threats, vulnerabilities, compliance, and incident trends.
- Presented insights through professional visualizations.
